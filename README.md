## üõ† Project Setup

### Prerequisites

- Node.js 18+  
- npm or pnpm  

### Install

```bash
git clone https://github.com/ensight-org/ensight-web.git
cd ensight-web
npm install
```

### Run locally

```bash
npm run dev
```

App will be available at: `http://localhost:3000`
